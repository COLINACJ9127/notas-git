1.- Entramos a la carpeta con "Git bash here"

git config --global user.email"jorge.colin@atl-technologies.com

Y el damos primeramente git init para poder iniciar un repositorio en la carpeta

2.- git init

Comando para dar inicio al repositorio

3.-git status

Esta opcion es para ver el status en qeu se encuentran los archivos,
podemos ver que despuesde haber hecho init, que ya se agrego un repositorio y los
archivos dentro del repositorio

4.- git diff (+ el nombdre del archivo)

it diff index.html

En esta sección podemos ver los cambios que se han hecho en los archivos, y exactamente que se ha hecho

Los cambios los mostrara de color verde en la consola

5.- gt add

Existen dos maneras de agrear los archivos modificados, la primera es uno por uno

por ejemplo;

git add index.html

o bien, si queremos agregar todos los archivos modificados, utilizamos el sigueinte comando;

git add .

Es muy importante darle la separación al punto, y de esta manera se agregaran todos los cambios realizados

6.- git commit

con git commit, mandamos a hacer todos los cambios que hemos hecho, y tecleamos el nombre que le daremos a ese cambio, para que quede guardado en el registro

con el sigueinte comando

git commit -m" nombre del cambio "

7.- .gitignore

Nota: las carpetas vacias las ignora

8.- git branch

Muestra todas las versiones alternativas

¿Como agregamos nuevas versioes alternativas?

Con el sigueinte comando

git branch + "nombre de la version alternativa"

git branch login es el nombre de nuestra version 1.1 lternativa

9.- git checkout

con git checkout, elejimos las versiones con las que queremos trabajar
pr poder entrar a la version1.1 que hemos creado, simplemento escribimos el comando

git checkout version1.1

Y automaticamente entraremos a trabajar con esa version de trabajo

Ahora los todos los cambios que guardemos se guardaran en esta version
